UIKit-View

UIImageView

  • 父类是UIView
  • UIImageView极其常用，功能比较专一：显示图片

常见属性

  1. image：显示的图片

    @property(nonatomic,retain) UIImage *image;

  2. animationImages：显示的动画图片

    @property(nonatomic,copy) NSArray *animationImages;

  3. animationDuration：动画图片的持续时间

    @property(nonatomic) NSTimeInterval animationDuration;

  4. animationRepeatCount：动画的播放次数（默认是0，代表无限播放

    @property(nonatomic) NSInteger      animationRepeatCount;

常见方法

  1. startAnimating：开始动画

    -(void)startAnimating;

  2. stopAnimating：停止动画

    • 使用场景：根据播放时间设置停止动画
      -(void)stopAnimating;

  3. isAnimating：是否正在执行动画

    -(BOOL)isAnimating;

  4. initWithImage：利用initWithImage创建出来的imageView的尺寸和传入的图片尺寸一样

    -(nullable instancetype)initWithImage:(UIImage *)image;
