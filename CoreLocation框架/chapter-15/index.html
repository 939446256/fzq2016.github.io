使用第三方框架定位

1. 主要原因

    因为使用CoreLocation框架进行获取用户位置信息, 是通过代理进行回调; 而第三方框架将"代理模拟"转换成为"block模式"; 使用起来比较方便, 而且额外增加了超时时间等功能.

2. 框架信息

    名称: locationManager
    地址: [link](https://github.com/intuit/LocationManager)

3. 使用方法

    参照该框架对应的 readME

4. 注意事项

    一般集成第三方框架到项目中, 请先确保该框架没有问题, 然后再向项目中集成
